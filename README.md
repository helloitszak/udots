# punctuation
Dotfiles and other fun.

This is my "works for me" dotfiles.

## what's in the box
* zsh
* vim
* emacs


## how do I get started
1. checkout the sourcecode to `~/.punctuation`
3. run `./install`
4. change your shell to zsh if it's not zsh already.
5. start doing things

## updating
1. `cd ~/.udots`
1. `git pull`
2. run `./install`
